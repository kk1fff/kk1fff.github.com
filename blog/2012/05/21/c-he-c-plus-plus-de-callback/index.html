C 和 C++ 的 Callback

May 21st, 2012 | Comments

C 的 Callback

Callback 把函數當成指標，在 C 裡面是這樣用

宣告時包含函數雛型，用 * 來宣告

傳入時使用取址運算子去取得函數指標

呼叫時 dereference 然後當函數使用

這樣的方法也可以用在 C++ 的靜態函數中（類別函數），但是不能用在物件函數。

C++ 的 Callback

C++ 有 this 指標，物件函數是需要有物件資料的。所以不能用傳統 C 的方式傳遞函數指標。C++ 物件函數的指標使用是這樣。

宣告的時候需要帶類別和雛型

傳送時用 & 運算子取址

呼叫時，需要使用實體來呼叫，實體可以是指標也可以是物件參考
